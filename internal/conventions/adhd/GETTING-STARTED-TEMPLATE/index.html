    ADHD Tips for This Template¶

    1. Time estimates ‚Äî Always include them (builds trust)
    2. Verify steps ‚Äî Every section should have a way to confirm success
    3. Collapsible hints ‚Äî Let users try before seeing answers
    4. Visual breaks ‚Äî Use horizontal rules between sections
    5. Cheat sheet ‚Äî Provide a copy-paste summary at the end
    6. Progressive complexity ‚Äî Start simple, build up
