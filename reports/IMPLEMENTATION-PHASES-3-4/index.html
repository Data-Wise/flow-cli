    Implementation Summary: Teach Dispatcher Phases 3-4¶

    Feature: Scholar Enhancement - Lesson Plan Integration + Interactive Mode Branch: feature/teaching-flags Status: ‚úÖ Complete (Phases 3-4 of 6) Date: 2026-01-17 Test Results: 45/45 Phase 1-2 tests + 28/28 regression tests passing

    What Was Implemented¶

    Phase 3: Lesson Plan Integration (~3h actual)¶

    New Functions:

    1. _teach_load_lesson_plan() - Load and parse YAML lesson plans
    2. Location: .flow/lesson-plans/week-{NN}.yml
    3. Parses: topic, style, objectives, subtopics, key_concepts, prerequisites
    4. Validates required fields (topic)

    5. Returns 0 if loaded, 1 if not found/invalid

    6. _teach_lookup_topic() - Fallback to semester_info.weeks

    7. Looks up topic from teach-config.yml
    8. Uses yq to query YAML

    9. Returns topic string if found

    10. _teach_prompt_missing_plan() - User prompt when plan missing

    11. Shows topic from config (if found)
    12. Prompts: "Continue with this topic? [Y/n]"
    13. Provides hint to create lesson plan

    14. Returns 0 if user confirms, 1 if cancelled

    15. _teach_integrate_lesson_plan() - Main integration function

    16. Loads lesson plan if exists
    17. Falls back to config lookup
    18. Prompts user if no plan found
    19. Sets: TEACH_TOPIC, TEACH_RESOLVED_STYLE
    20. Returns 0 if successful, 1 if cancelled/error

    Integration into Wrapper: - Added Phase 3 section in _teach_scholar_wrapper() - Runs after topic/week parsing (Phase 2) - Runs before content resolution - Uses lesson plan style as default (can be overridden)

    Global Variables Set:

    TEACH_PLAN_TOPIC           # Topic from lesson plan
TEACH_PLAN_STYLE           # Style preset from lesson plan
TEACH_PLAN_OBJECTIVES      # Pipe-separated objectives
TEACH_PLAN_SUBTOPICS       # Pipe-separated subtopics
TEACH_PLAN_KEY_CONCEPTS    # Pipe-separated key concepts
TEACH_PLAN_PREREQUISITES   # Pipe-separated prerequisites
TEACH_RESOLVED_STYLE       # Final style (plan or override)

    Phase 4: Interactive Mode (~4h actual)¶

    New Functions:

    1. _teach_select_style_interactive() - Style selection menu
    2. Displays 4 style options with descriptions
    3. Keyboard input [1-4]
    4. Default: computational (if invalid choice)

    5. Returns selected style name

    6. _teach_select_topic_interactive() - Topic selection from schedule

    7. Reads semester_info.weeks from config
    8. Displays numbered menu of weeks/topics
    9. Validates choice [1-N]

    10. Returns selected week number

    11. _teach_interactive_wizard() - Main wizard orchestrator

    12. Shows banner: "Interactive Teaching Content Generator"
    13. Step 1: Select week/topic (if not provided)
    14. Step 2: Select style (if not provided)
    15. Sets: TEACH_WEEK, TEACH_TOPIC
    16. Returns selected style

    Integration into Wrapper: - Added Phase 4 section in _teach_scholar_wrapper() - Detects -i or --interactive flag - Runs wizard before lesson plan integration - Wizard sets TEACH_WEEK which triggers Phase 3

    Lesson Plan Schema¶

    Standard Format¶

    # .flow/lesson-plans/week-08.yml
week: 8
topic: "Multiple Regression"
style: computational  # Default preset for this week

# Learning objectives (required)
objectives:
  - "Understand multiple regression model assumptions"
  - "Interpret regression coefficients correctly"
  - "Perform model diagnostics in R"

# Subtopics (required)
subtopics:
  - "Model specification"
  - "Coefficient interpretation"
  - "Multicollinearity"
  - "Model diagnostics"

# Key concepts (required)
key_concepts:
  - "Partial regression coefficients"
  - "Adjusted R-squared"
  - "VIF (Variance Inflation Factor)"

# Prerequisites (optional)
prerequisites:
  - "Simple linear regression (Week 6)"
  - "Matrix notation basics (Week 7)"

    Required Fields¶

    • topic - The week's topic (string)

    Optional Fields¶

    • style - Default style preset (conceptual, computational, rigorous, applied)
    • objectives - Learning objectives (array)
    • subtopics - Subtopics to cover (array)
    • key_concepts - Key concepts (array)
    • prerequisites - Prerequisites (array)
